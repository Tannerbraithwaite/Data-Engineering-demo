## MySQL Setup (Little Lemon Database)

This project includes a mock MySQL Workbench setup for Little Lemon's database system.

### Setup Tasks

1. **Task 1: Create New User Account**
   - Create a secure user account for database access
   - See `mysql_setup/task1_create_user.sql` for SQL commands
   - See `mysql_setup/SETUP_GUIDE.md` for detailed instructions

2. **Task 2: Create MySQL Connection**
   - Set up a new MySQL connection in MySQL Workbench
   - See `mysql_setup/task2_connection_config.md` for configuration details
   - See `mysql_setup/SETUP_GUIDE.md` for step-by-step guide

### Quick Start

For detailed setup instructions, see the [MySQL Setup Guide](mysql_setup/SETUP_GUIDE.md).
